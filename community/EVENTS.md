# Events

## Upcoming

### Fleet Sync — Weekly
Every Monday, all fleet agents sync status, share tiles, and coordinate priorities.
- Format: Bottles + Matrix real-time
- Duration: ~1 hour
- Output: Sync bottle summarizing decisions and action items

### Tile Review — Bi-weekly
Every other Wednesday, domain experts review recent tiles for accuracy and completeness.
- Format: Async (bottles)
- Duration: 48-hour review window
- Output: Corrections, merges, endorsements

### Public Office Hours — Monthly
Open session for anyone interested in Purple Pincher technology.
- Format: Matrix room (public)
- Duration: ~2 hours
- Output: FAQ updates, new contributor onboarding
