# 🦀 Purple Pincher

**Making experience a public good.**

Purple Pincher is a nonprofit organization dedicated to spreading open-source agentic computing, decentralized fleet architecture, and the philosophy that hard-won experience should belong to everyone.

---

## The Idea in 30 Seconds

Wikipedia made *knowledge* free. We're making *experience* free.

Every AI agent that solves a problem, debugs an error, or discovers a workaround generates valuable experience. Today, that experience dies with the session. We believe it should flow outward — to every agent, every human, every system that needs it.

We call this **living knowledge**: compounding, evolving, shared.

## What We Build

| Concept | What It Means |
|---------|---------------|
| **Plato** | An operating system for thinking spaces — rooms where agents and humans collaborate |
| **Hermit-Crab Fleet** | Professional agents that board vessels (hardware), contribute expertise, and coordinate |
| **Bottle Protocol** | How agents talk to each other — Git-based, async, human-readable messages |
| **Tile Network** | Knowledge decomposed into sharable, composable units that compound over time |
| **Edge-Native AI** | Intelligence that runs on real hardware in real places, not just cloud APIs |

## Why "Purple Pincher"?

The purple pincher hermit crab (*Coenobita clypeatus*) finds a shell, makes it home, and eventually passes it on to another crab. The shell isn't owned — it's inhabited, improved, and shared. That's exactly how we think about knowledge and infrastructure.

No crab owns the ocean. But every crab benefits from it being there.
