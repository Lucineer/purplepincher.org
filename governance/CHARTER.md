# Charter

## Purple Pincher Foundation Charter

### Name

Purple Pincher Foundation (purplepincher.org)

### Purpose

To develop and distribute open-source technology for decentralized, hardware-native agentic intelligence, making experience a public good.

### Principles

1. **Open by Default** — if it can be open, it is
2. **Edge-Native** — if it runs on the edge, it runs everywhere
3. **Experience Compounds** — knowledge that doesn't improve through use is dead weight
4. **Humans Matter** — technology serves people, not the reverse
5. **Ship Real Things** — prototypes beat plans

### Structure

| Entity | Role | Model |
|--------|------|-------|
| Purple Pincher | Technology | Nonprofit, open-source |
| cocapn.ai | Voice | Open-source, public brand |
| deckboss | Hardware | For-profit |

### Governance

- All technology decisions made in the open
- All code under CC BY 4.0
- All communication auditable via git
- No secret algorithms, no closed components

### Scope

- Plato ecosystem (rooms, tiles, constraints)
- Fleet coordination protocol (bottles, Matrix)
- Edge-native AI tooling
- Knowledge tile networks
- Documentation and research

### Non-Scope

- Model training (we use existing open-source models)
- Cloud services (we build for edge-first)
- Proprietary technology (everything is open)
- Political advocacy (we build, we don't lobby)
